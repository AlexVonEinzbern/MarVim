# MarVim

## Requirements
- Nvim >= 0.8.3
- Some [Nerd Font](https://www.nerdfonts.com/) on your terminal

## Installation on Linux

```
git clone https://github.com/AlexVonEinzbern/MarVim.git ~/.config/nvim --depth 1 && nvim
```
## Uninstall on Linux

```
rm -rf ~/.config/nvim
rm -rf ~/local/share/nvim
```
